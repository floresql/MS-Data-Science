# MS in Data Science | Regis University

This repository documents my academic journey and technical projects while pursuing my **Master of Science in Data Science** at Regis University. 

## Program Overview
* **Institution:** Regis University (Anderson College of Business and Computing)
* **Degree:** M.S. in Data Science
* **Focus:** Data Engineering, Predictive Modeling, and Data Ethics

## Curriculum & Projects

### Core Foundations
* **MSDS 600: Intro to Data Science** – Foundational workflows in Python and R.
* **MSDS 610: Data Engineering** – ETL processes, database management, and data wrangling.
* **MSDS 650: Data Analytics** – Statistical discovery and business intelligence.

### Advanced Analytics
* **MSDS 660: Statistical Methods** – Experimental design and hypothesis testing.
* **MSDS 680: Machine Learning** – Supervised/unsupervised learning and model evaluation.
* **MSDS 670: Data Visualization** – Storytelling with data using Tableau and ggplot2.

### Ethics & Capstone
* **MSDS 640: Ethics, Privacy, & Social Justice** – Analyzing bias and privacy in algorithmic decision-making.
* **MSDS 692/696: Data Science Practicum** – Industry-aligned capstone project.

## Technical Toolkit
* **Languages:** Python (Pandas, Scikit-Learn, NumPy), R (Tidyverse), SQL.
* **Tools:** Jupyter Notebooks, RStudio, Git/GitHub, Tableau.
* **Environments:** AWS, Hadoop, Spark.
